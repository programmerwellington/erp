# Partial ERP System

This code aims to demonstrate my knowledge in:
- Java,
- DAO (Data Access Object),
- JDBC,
- association class (Erp_Order_Product) that associates the classes (Erp_Order) and (Erp_Product);
- auxiliary class (Erp_Order_Product_Pk) that resolves the adaptation of the Composite Primary Keys of the MySQL Relational Database paradigm with the multi-paradigm Object-Oriented/Functional Programming of Java;
- connection to MySQL;
- Exceptions and
- Tests.

This is a partial **ERP** backend system, developed in Java; connecting to MySQL in the CLIENT, PRODUCT, and ORDER tables; using the concepts of DAO (Data Access Object) as a contract-interface and JDBC implementation.

## Technologies Used

* **Language:** Java (JDK 25+)
* **API:** JDBC
* **Database:** MySQL
